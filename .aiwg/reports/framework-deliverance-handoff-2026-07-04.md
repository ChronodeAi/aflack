## What the framework now knows

- Optimize for engineered virality, not artifact volume.
- Use personas when they improve reach or monetization; allow persona-free formats when benchmark evidence is stronger.
- Keep Postiz as the scheduling/posting boundary.
- Keep the raw event store as source of truth.
- Run Jarvis-style orchestration as human-gated direction, not unsupervised account automation.
- Block GTA6 pre-release Rockstar footage and same-seed derivatives.
- Treat real-event Tier 3 ingestion as an explicit operator risk override.
- Prioritize portable funnels and owned audience assets over platform-payout dependency.

## Construction handoff

The next construction pass should build only the smallest measurable loop:

1. Load active insights and open gates.
2. Choose one package: Vice Signal revision, Loadout Lab affiliate package, or Postiz draft preview.
3. Generate/validate script, prompt, CTA, disclosure, and economics estimate.
4. Stop before paid generation or public publish unless explicitly approved.
5. Ingest results and cost data before scaling.

## Frameworkization gate

Do not promote this into a full reusable AIWG framework until it has real operating data:

- at least one complete publish/result/economics loop,
- positive or instructive contribution-margin evidence,
- a repeated winner pattern,
- stable compliance outcomes,
- memory lessons that improve the next package.
