# Workflow: claude-video-production

1. Retrieve active insights.
2. Create hook options using `hook-authoring`.
3. Create Claude Code video builder prompt.
4. Generate pre-generation package only.
5. Convert storyboard into a beat-level prompt pack using `higgsfield-asset-prompting`.
6. Run compliance, virality review, and prompt-quality review.
7. Ask for spend approval.
8. Only then generate media.
9. Preview Postiz draft.
10. Ask for publish approval.
