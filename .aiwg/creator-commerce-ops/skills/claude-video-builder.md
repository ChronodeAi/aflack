# Skill: claude-video-builder

Purpose: use Claude Code as the pre-generation video director.

Required package sections:
1. active benchmark insight citations,
2. hook options,
3. selected hook rationale,
4. timestamped script,
5. storyboard/shot list,
6. text-only Higgsfield prompt,
7. title/description/pinned comment,
8. compliance checklist,
9. measurable hypothesis,
10. spend gate.

Every package must cite at least one active benchmark insight before it can
reach spend review. Prefer the highest-confidence insights from the current
niche, then fall back to durable cross-niche hook, structure, CTA, and funnel
insights.

Required CTA/funnel path:
- one clear viewer action,
- one destination for conversion,
- a lead magnet, link hub, or owned offer that explains what happens next.

For GTA6-adjacent packages, wording must not imply leaked access, official
gameplay access, or Rockstar/Take-Two affiliation. Rewrite those angles into
clearly labeled rumor, speculation, commentary, or creator-workflow framing.

No generation or Postiz submission is allowed from this skill.
